Setup

WiFi Network Configuration Data

Enter the SSID and the password of the router you want to connect.

Static IP settings

If you want to assing a static IP address fill out the following information. All addresses have to by given in IPv4 format (xxx.xxx.xxx.xxx).
Leave the fields empty to rely on DHCP to obtain an IP address.

Telnet Server Port

The Vitotronic WiFi Interface will listen at this port for incoming telnet connections.

UDP Server Port

The 1-wire UDP port is fixed to: 12345

Reconnect timeout

The Vitotronic WiFi Interface will try to connect (timeout) sec. after the reboot of the router.

1-wire interval

In case of 1-wire temperature sensors are connected, send the temperatures every (interval) sec. (between 20 and 3600 s).

Fields marked by (*) are mandatory.
